## L’essentiel

Royal Brompton Hospital, part of Guy’s and St Thomas’ NHS Foundation Trust (London, UK) has become the first hospital in the UK to install the Magnetom Cima.X magenetic resonance imaging (MRI) system (Siemens Healthineers). Magnetom Cima.X enables novel heart scanning techniques using diffusion-weighted imaging.
